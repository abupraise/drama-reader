# Sound effects for DramaReader

Place your manually downloaded audio files in this folder using these exact filenames when possible:

- `thunder.mp3`
- `rain.mp3`
- `crowd.mp3`
- `suspense_sting.mp3`
- `heartbeat.mp3`
- `wind.mp3`
- `applause.mp3`
- `door_creak.mp3`
- `footsteps.mp3`
- `dramatic_chord.mp3`

The app also accepts `.wav` or `.ogg` with the same base names.

## Free Creative Commons sound sources

You can find suitable CC-licensed effects from:

- Freesound: https://freesound.org/
- Pixabay sound effects: https://pixabay.com/sound-effects/
- Mixkit free sound effects: https://mixkit.co/free-sound-effects/

## Suggested search terms

Use these search phrases:

- thunder rumble
- heavy rain ambience
- crowd murmur
- suspense sting cinematic
- heartbeat dramatic
- wind ambience
- applause audience clap
- door creak wooden door
- footsteps hallway
- dramatic chord impact

## Important

- Verify the license before downloading.
- Prefer MP3 files for simplicity.
- Keep each sound relatively short for smoother mixing.
- If any sound file is missing, DramaReader will skip it automatically and continue.
